# 주조 공정 최적화 및 불량률 예측 프로젝트

본 프로젝트는 '주조 공정 최적화 데이터셋'을 활용하여, 데이터의 잠재적 가치를 시각화를 통해 발견하고 머신러닝 모델을 통해 실용적인 분석을 수행하는 데 중점을 두었습니다. EDA(탐색적 데이터 분석) 과정에서 다양한 시각화 기법을 사용하여 데이터의 특성과 변수 간의 상관관계를 파악했으며, 이를 바탕으로 데이터 전처리 및 피처 엔지니어링을 수행했습니다. 최종적으로 분류 모델을 구축하여 주조 공정의 불량률을 0.99의 정확도로 예측하는 성과를 달성했습니다.

1. 데이터 시각화 능력 (EDA 및 인사이트 도출)
데이터에 대한 깊은 이해는 정확한 분석의 첫걸음입니다. 본 프로젝트에서는 다음과 같은 시각화 작업을 통해 데이터의 특성을 파악하고 분석 방향을 설정했습니다.

변수 분포 확인: Pandas, Matplotlib, Seaborn 라이브러리를 활용하여 각 공정 변수(온도, 속도, 압력 등)의 데이터 분포(히스토그램, 커널 밀도 추정)를 시각화하여 데이터가 특정 범위에 편중되어 있지는 않은지 확인했습니다.

상관관계 분석: 주요 변수 간의 상관관계를 파악하기 위해 **히트맵(Heatmap)**을 시각화했습니다. 이를 통해 불량률(Target)과 높은 상관관계를 보이는 변수들을 1차적으로 식별했습니다.

이상치 탐지: **박스 플롯(Box Plot)**을 사용하여 각 변수의 사분위수와 이상치(Outlier)를 시각적으로 확인하고, 이러한 값들이 모델에 미칠 영향을 판단하여 전처리 방향을 결정했습니다.

다변수 관계 파악: **페어 플롯(Pair Plot)**을 사용하여 여러 변수 간의 관계를 한눈에 파악하고, 정상 데이터와 불량 데이터가 어떤 변수 조합에서 잘 분리되는지 탐색했습니다.

2. 데이터 활용 및 분석 능력 (전처리 및 모델링)
시각화를 통해 얻은 인사이트를 바탕으로 데이터를 실질적으로 활용하고 분석하여 가치 있는 결과를 도출했습니다.

데이터 전처리:

시각화 분석에서 발견한 결측치 및 이상치를 통계적 기법(예: IQR)을 기반으로 처리했습니다.

모델의 성능 향상을 위해 수치형 데이터에 대해 StandardScaler를 적용하여 데이터의 스케일을 표준화했습니다.

모델 구축 및 활용:

이 프로젝트를 '정상'(0.0)과 '불량'(1.0)을 예측하는 이진 분류(Binary Classification) 문제로 정의했습니다.

데이터의 특성에 맞는 머신러닝 모델(예: Logistic Regression, Random Forest 등)을 선정하여 학습을 진행했습니다.

모델 성능 평가 및 분석:

단순히 정확도(Accuracy)만 보는 것이 아니라, 불균형 데이터(정상 17379개, 불량 592개)의 특성을 고려하여 Classification Report를 중점적으로 분석했습니다.

Precision, Recall, F1-Score를 통해 모델이 불량(1.0) 케이스를 얼마나 잘 예측하는지 정밀하게 평가했으며, 0.99라는 높은 전체 정확도와 더불어 불량 예측(F1-score: 0.90)에서도 유의미한 성과를 확인했습니다.# -
